You can use the BA CLI to run the entire BA stack locally on your machine, by running `supabase init` and then `supabase start`. To install the CLI, see the [installation guide](/docs/guides/cli/getting-started#installing-the-supabase-cli).

The BA CLI provides tools to develop your project locally, deploy to the BA Platform, handle database migrations, and generate types directly from your database schema.
