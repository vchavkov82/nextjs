## Common Commands

### List Runners

```bash
runners list
```

### Create Runner

```bash
runners create --type linux --size medium
```

### Update Runner

```bash
runners update <runner-id> --size large
```

### Delete Runner

```bash
runners delete <runner-id>
```

### View Logs

```bash
runners logs <runner-id>
```

### Configure

```bash
runners config set <key> <value>
runners config get <key>
```

## Configuration

CLI configuration can be stored in:

- Environment variables
- Configuration files
- Command-line flags

## Authentication

The CLI uses authentication tokens for API access. Set your token:

```bash
runners auth login
```
