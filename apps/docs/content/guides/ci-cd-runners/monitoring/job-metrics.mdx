title: 'Job Metrics'

Monitor job performance and resource usage with CI/CD Runners job metrics.

## Overview

Job metrics provide detailed insights into job execution, including execution time, resource usage, and success rates.

## Available Metrics

Job metrics include:

- **Execution time**: Time taken for jobs to complete
- **Resource usage**: CPU, memory, disk usage
- **Success rate**: Percentage of successful jobs
- **Queue time**: Time jobs wait before execution
- **Cost per job**: Cost breakdown per job

## Viewing Metrics

View job metrics through:

- Dashboard interface
- API endpoints
- Monitoring integrations
- Export to external tools

## Examples

### Execution Time

Track average execution time:

```
Average execution time: 5.2 minutes
P95 execution time: 12.4 minutes
P99 execution time: 18.7 minutes
```

### Resource Usage

Monitor resource utilization:

```
Average CPU usage: 65%
Average memory usage: 4.2 GB
Average disk usage: 15 GB
```

## Best Practices

- Monitor execution times
- Track resource utilization
- Set up alerts for anomalies
- Optimize based on metrics
