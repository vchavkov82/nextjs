Learn how to enable cost reporting functionality for CI/CD Runners.

## Overview

Cost reporting provides detailed insights into spending on CI/CD Runners, helping you optimize costs.

## Cost Breakdown

Cost reports include:

- **Total costs**: Overall spending
- **By runner type**: Costs per runner type
- **By environment**: Costs per environment
- **By job**: Costs per job type
- **Resource costs**: Infrastructure costs
- **Trends**: Cost trends over time

## Viewing Reports

View cost reports through:

- Dashboard interface
- API endpoints
- Scheduled reports
- Export to external tools

## Cost Optimization

Use cost reports to:

- Identify cost drivers
- Optimize runner sizing
- Use spot instances effectively
- Right-size pools
- Monitor cost trends

## Best Practices

- Review costs regularly
- Set cost budgets
- Monitor cost trends
- Optimize based on reports
- Share reports with stakeholders
