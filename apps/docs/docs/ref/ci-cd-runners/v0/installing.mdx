## Installation

To install CI/CD Runners v0.8.0, follow these steps:

1. Prepare your infrastructure with the required resources
2. Deploy the runner binary to your environment
3. Configure the runner with your CI/CD provider credentials
4. Verify the installation by running a test pipeline

For detailed installation instructions specific to your platform, see the [Self-Hosting Guide](/docs/guides/ci-cd-runners).

<Admonition type="warning">

This is version 0.8.0 (legacy). For current users, please consider upgrading to version 1.0.0 for improved stability and additional features.

</Admonition>
