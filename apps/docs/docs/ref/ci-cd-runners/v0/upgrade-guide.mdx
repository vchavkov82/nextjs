## Upgrade Guide

### Upgrading from 0.8.0 to 1.0.0

The upgrade from version 0.8.0 to 1.0.0 is designed to be straightforward with minimal downtime.

#### Backward Compatibility

Most configurations from version 0.8.0 are compatible with version 1.0.0. You may need to update:
- Some environment variable names
- Custom resource specifications
- Monitoring configurations

#### Upgrade Steps

1. **Backup your current configuration**
   ```bash
   cp -r /path/to/ci-cd-runners /path/to/ci-cd-runners.backup
   ```

2. **Stop the current runner service**
   ```bash
   sudo systemctl stop ci-cd-runners
   ```

3. **Deploy version 1.0.0**
   ```bash
   # Follow the installation guide for version 1.0.0
   ```

4. **Verify the upgrade**
   ```bash
   ci-cd-runners --version
   ci-cd-runners health-check
   ```

5. **Restart services**
   ```bash
   sudo systemctl start ci-cd-runners
   ```

#### Testing

Before fully migrating workloads, test your critical pipelines with the new version.

For detailed migration instructions, contact support or refer to the [Self-Hosting Guide](/docs/guides/ci-cd-runners).
