## Version 0.8.0 Release Notes

### Features

- Self-hosted CI/CD runner infrastructure
- Support for multiple CI/CD platforms
- Docker and container build support
- Job scheduling and resource management
- Basic monitoring and logging capabilities

### Known Limitations

- Limited real-time metrics
- Basic resource allocation options
- Community support only

### Upgrade Path

Users on version 0.8.0 are encouraged to upgrade to version 1.0.0 for:
- Enhanced stability and performance
- Additional platform support
- Improved resource optimization
- Professional support options

For upgrade instructions, see the [Upgrade Guide](/docs/guides/ci-cd-runners).
