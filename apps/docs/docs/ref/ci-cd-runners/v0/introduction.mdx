This reference documents the CI/CD Runners API and usage patterns. CI/CD Runners allow you to run your CI/CD workflows on your own infrastructure, providing better performance, cost savings, and control over your build environment.

<Admonition type="note">

CI/CD Runners is a self-hosted solution for running continuous integration and deployment workflows. Version 0.8.0 represents the legacy release.

For the latest features and improvements, please upgrade to version 1.0.0 or newer.

</Admonition>
