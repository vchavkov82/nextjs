# CI/CD Runners v0.x Reference

Legacy version - for new projects, use [CI/CD Runners v1](/reference/ci-cd-runners).

## Overview

CI/CD Runners v0.x is the previous release. This documentation is provided for existing projects still using v0.

**Note:** v0 is no longer actively maintained. We recommend upgrading to v1 for:
- Better performance
- Enhanced features
- Security updates
- Active support

## Upgrading to v1

See the [Upgrade Guide](/guides/ci-cd-runners/guides/upgrade) for migration instructions from v0 to v1.

For v0.x-specific documentation, refer to the legacy guides section.
