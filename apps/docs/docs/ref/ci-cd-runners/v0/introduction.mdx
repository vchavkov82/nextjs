# CI/CD Runners v0 Reference

Legacy version - for new projects, use **[CI/CD Runners v1](/reference/ci-cd-runners)**.

## About v0

CI/CD Runners v0 is the previous release. This documentation is provided for existing projects still using v0.

## Upgrading to v1

We recommend upgrading to v1 for:
- Better performance (up to 40x faster)
- Enhanced features and improvements
- Security updates
- Active support and maintenance

See the **[Upgrade Guide](/docs/guides/ci-cd-runners/guides/upgrade)** for migration instructions.

## Current Version

For the latest features and improvements, see **[CI/CD Runners v1](/reference/ci-cd-runners)** and the **[comprehensive guides](/docs/guides/ci-cd-runners)**.
