This reference documents the CI/CD Runners API and usage patterns. CI/CD Runners allow you to run your CI/CD workflows on your own infrastructure, providing better performance, cost savings, and control over your build environment.

<Admonition type="note">

CI/CD Runners is a self-hosted solution for running continuous integration and deployment workflows. This is the latest release with full feature support and regular updates.

For legacy versions, see [version 0.8.0](/reference/ci-cd-runners/v0).

</Admonition>
