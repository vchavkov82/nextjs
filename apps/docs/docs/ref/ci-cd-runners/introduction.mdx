# CI/CD Runners Reference

This is the reference documentation for CI/CD Runners v1.0. For detailed guides and tutorials, see the [CI/CD Runners Guides](/guides/ci-cd-runners).

For legacy versions, see [CI/CD Runners v0](/reference/ci-cd-runners/v0).

## Overview

CI/CD Runners is a self-hosted solution for running continuous integration and deployment workflows on your own infrastructure, providing better performance, cost savings, and control over your build environment.
