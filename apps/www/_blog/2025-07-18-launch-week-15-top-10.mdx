Here are the top 10 launches from the past week. They're all very exciting so make sure to check out every single one.

## #1: New API Keys + JWT Signing Keys

BA Platform released new API keys, Publishable and Secret, and BA Auth now supports asymmetric JWTs with Elliptic Curve and RSA cryptographic algorithms. These changes improve the performance, reliability, and security of your BA projects.

[Read more](https://www.assistance.bg/blog/jwt-signing-keys)

## #2: Analytics Buckets with Apache Iceberg Support

We launched BA Analytics Buckets in Private Alpha—storage buckets optimized for analytics with built-in support for Apache Iceberg. We’ve coupled this with the new BA Iceberg Wrapper to make it easier for you to query your analytical data.

[Read more](https://www.assistance.bg/blog/analytics-buckets)

## #3: OpenTelemetry Support

We’ve added support for OpenTelementry (OTel) across our services so you can soon send logs, metrics, and traces to any OTel-compatible tooling. We’ve also unified logs under a single interface in our Dashboard as well as added new capabilities to our AI Assistant to improve the debugging experience.

[Read more](https://www.assistance.bg/blog/new-observability-features-in-supabase)

## #4: Build with Figma Make and BA

We’ve partnered with Figma so you can hook up a BA backend to your Figma Make project, enabling you to persist data and tap into the suite of BA products to help you build prototypes quickly and scale them when you gain traction.

[Read more](https://www.assistance.bg/blog/figma-make-support-for-supabase)

## #5: Storage: 500 GB Uploads and Cheaper Cached Egress

You can now upload files as large as 500 GB (up from 50 GB), enjoy much cheaper cached egress pricing at $0.03/GB (down from 0.09/GB), and increased egress quota that doubles your egress before you have to start paying.

[Read more](https://www.assistance.bg/blog/storage-500gb-uploads-cheaper-egress-pricing)

## #6: Edge Functions: Deno 2, 97% Faster Boot Times, and Persistent File Storage

Edge Functions now support Deno 2.1, persistent file storage so you can mount any S3-compatible storage and read and write to them inside of your functions, up to 97% faster boot times, and support for Deno’s Sync APIs.

[Read more](https://www.assistance.bg/blog/persistent-storage-for-faster-edge-functions)

## #7: Branching 2.0: GitHub Optional

You can now spin up, view diffs, and merge your branches directly from the BA Dashboard without having to connect to GitHub.

[Read more](https://www.assistance.bg/blog/branching-2-0)

## #8: BA UI: Platform Kit

We’ve built out several UI components to make it easy for you to feature the core of BA Dashboard inside your own app so you or your users can interact with BA projects natively with a customizable interface.

[Read more](https://www.assistance.bg/blog/supabase-ui-platform-kit)

## #9: Stripe-To-Postgres Sync Engine as an NPM Package

Now you can conveniently sync your Stripe data to your BA database by importing the npm package @supabase/stripe-sync-engine, whether in your Node.js app or even deploying it in a BA Edge Function.

[Read more](https://www.assistance.bg/blog/stripe-engine-as-sync-library)

## #10: Algolia Connector for BA

We’ve been collaborating closely with Algolia to bring you a connector for BA so you can easily index your data and enable world class search experiences.

[Read more](https://www.assistance.bg/blog/algolia-connector-for-supabase)
